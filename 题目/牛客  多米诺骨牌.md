## 描述

小利同学摆放了 $n$ 张多米诺骨牌，第 $i$ 张骨牌高为 $h_{i}$​，摆放在位置 $x_{i}$​。该骨牌向后倒塌时会使位于 $[x_{i},h_{i}+x_{i}]$区间(包含 $x_{i},h_{i}+x_{i}$​)的骨牌全部倒塌。  
  
小利同学摆放完骨牌就开心的出去玩耍了。结果小金同学发现了这些骨牌，他想趁小利不在，选择**不超过** $m$ 张骨牌并依次向后推倒，请算算**最多**会有多少张骨牌倒塌。  
  
保证任意位置上至多存在一张骨牌。
### 输入描述：

第一行给出一个整数 $T(1≤T≤5×10^4)$ ，表示数据组数。  

每组数据第一行给出两个整数 $n,m(1≤m≤n≤2×10^5,∑n≤2×10^5)$ ，分别表示骨牌总数与最多可推倒骨牌数。  

第二行依次给出 $n$ 个整数，表示骨牌高度 $h_{i}(1≤h_{i}≤10^9)$ 。  

第三行依次给出 $n$ 个不同整数，表示骨牌位置 $x_{i}(1≤x_{i}≤10^9)$ 。
### 输出描述：

每组测试数据输出一个整数，表示骨牌最多倒塌数。

## 示例1

输入：
```plaintext
3
6 1
1 1 1 3 2 1
4 3 2 7 9 11
6 2
1 1 1 3 2 1
4 3 2 7 9 11
5 4
1 4 1 1 2
1 2 3 6 8
```

输出：
```plaintext
3
6
5
```
### 思路：
1. 封装为结构体，按位置 x 升序排列
2. 区间分组（用大顶堆）若 A 的覆盖区间 $[x_{a},x_{a} + h_{a}]$ 大于物体 B 的位置 $x_{b}$ 则A和B属于同一组，遍历所有物品，将组的数量 t 存入到 大顶堆中
3. 选取最大 m 组求和
### code：
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;
typedef long long ll;

//将骨牌封装成结构体，并重载运算符使骨牌按x升序排列
struct Dominos {
    ll x,h;
    bool operator < (const Dominos& Domino) const{
        return x < Domino.x;
    }
};
void solve() {
    int n,m;
    cin >> n >> m;
    vector<Dominos> a(n);
    for (int i = 0 ; i < n ;  ++i) {
        cin >> a[i].h;
    }
    for (int i = 0 ; i < n ; ++i) {
        cin >> a[i].x;
    }
	
	//这里的sort是直接按照重载的定义，只按x的升序排列
    sort(a.begin(),a.end());
    
    priority_queue<int> q; //用大顶堆为了更好进行区间合并，同时排好序
    ll cmp = a[0].x + a[0].h,cnt = 1;//这里的cmp是该区间的最远覆盖位置，cnt为区间长度
    //遍历每个骨牌，分组
    for (int i = 0 ; i < n ; ++i) {
	    //遍历到最后一个直接保存个数，最后一个已经加到cnt上了
        if (i == n - 1) {
            q.push(cnt);
            break;
        }
        if (cmp >= a[i + 1].x) {
            ++cnt; //如果cmp比下一个的位置要大，cnt+1
            cmp = max(cmp,a[i + 1].x + a[i + 1].h); //更新cmp
        }
        else {
	        //否则直接保存区间长度
            q.push(cnt);
            cnt = 1;
            cmp = a[i + 1].x + a[i + 1].h; //更新cmp
        }
    }

    ll ans = 0;
    for (int i = 0 ; i < m ; ++i) { //选取前m个最大的区间长度
        if (q.empty()) {
            break;
        }
        ans += q.top();
        q.pop();
    }
    cout << ans << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
