*Algorithm :* [[Prim算法 (最小生成树)]]

We wish to construct an electricity generation and transmission network in a cluster of $N$ islands. To this end, we have calculated a cost $p(x)$ to (potentially) build a generating station on each island $x$, as well as a cost $c(x,y)$ to build an energy transmission line between any two islands $x$ and $y$. An island is connected to the grid if it has a generating station built on it, or if there is a sequence of power transmission lines connecting it to an island that has a generating station. Our goal is to connect all $N$ islands to the grid, minimizing the total network construction cost.

**Input**

The first line of the input contains an integer $N$ which gives the number of islands in the cluster.

The next $N$ lines of the input are sequences of $N$ numbers that form an $N$\-by-$N$ array $A$, such that each diagonal entry $A[x,x]$ corresponds to the cost $p(x)$ to build generating station on island $x$ (that is, $'A[x,x] = p(x)'$), while each non-diagonal entry $A[x,y]$ corresponds to the cost $c(x,y)$ to build an energy transmission line between two islands $x$ and $y$ (that is, $'A[x,y] = c(x,y)'$).

**Output**

A single integer that gives the total construction cost of the network.

**Example**

Input
```
4
10 8 10 14
8 15 9 13
10 9 12 12
14 13 12 11
```

Output
```
38
```

**Note**

Limits

\- $1 \leq N \leq 1000$

\- $1 \leq A[x,y] \leq 10000, \forall 1 \leq x,y \leq N$
### 思路：
1. 该问题是要求最小成本 = n个节点的最小生成树的总权重
2. Prim算法本质上是一个贪心算法，适配稠密图
3. 每次选择岛屿都选成本最小的岛屿，同时更新其他岛屿的成本

### ACcode
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void solve() {
    int n;
    cin >> n;

    vector<vector<int>> grid(n,vector<int>(n));
    for (int i = 0 ; i < n ; ++i) {
        for (int j = 0 ; j < n ; ++j) {
            cin >> grid[i][j];
        }
    }

    //vis数组：标记岛屿是否已加入最小生成树（MST），初始为全0（未加入）
    //cur数组：prim算法的距离数组，cur[j]表示未加入MST的岛屿j到已加入MST到的岛屿之间的最小距离
    vector<int> cur(n),vis(n,0);
    for (int i = 0 ; i < n ; ++i) cur[i] = grid[i][i];

    int ans = 0;
    for (int i = 0 ; i < n ; ++i) { //选n个节点加入MST
        int chosen = -1;
        for (int j = 0 ; j < n ; ++j) {
            //选未加入MST，cur最小的节点作为chosen
            if (!vis[j] && (chosen == -1 || cur[j] < cur[chosen])) {
                chosen = j;
            }
        }

        //将chosen加入MST，累加边权更新答案
        ans += cur[chosen];
        vis[chosen] = 1; //标记为已访问

        //更新所有未加入MST的节点的cur值
        for (int j = 0 ; j < n ; ++j) {
            if (!vis[j]) {
                cur[j] = min(cur[j],grid[chosen][j]);
            }
        }
    }
    cout << ans << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
