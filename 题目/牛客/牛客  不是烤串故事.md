*Algorithm :* [[字符串]]
## 描述

   小红有两个长度为 $n$ 的字符串 $s$ 和 $t$ ，我们定义下标从 1 开始，现在你可以选取字符串 $s$ 的前 $i$ 个字符 $s_{1}s_{2}\dots s_{i}$​，然后将这一部分反转后与剩余部分拼接，得到 $s_{i}'$。

请你找到每一个翻转前缀 $s_{i}'$​ 与字符串 $t$ 的 $max\_len_{i=1}^{n}~\left \{lcp(s_{i}',t)\right \}$ ，即长度最长的 $lcp(s_{i}',t)$ 。在这里，$lcp$ 代表最长公共前缀。

好吧，这其实并不难，作为神秘的 $F$ 题，你同时需要输出满足上述条件的最小的 $i$ 。

 在本题中，反转即为将字符串绕中心字符前后反转，具体地说，设字符串为 $s_{1}s_{2}\dots s_{n-1}​s_{n}$ ，反转后得到 $s_{n}s_{n-1}​\dots s_{2}s_{1}$​。  

### 输入描述：

 每个测试文件均包含多组测试数据。第一行输入一个整数 $T(1 \leq T \leq 100)$ 代表数据组数，每组测试数据描述如下：  
         第一行输入一个整数 $n(1\leq n\leq 10^6)$ 代表字符串长度。  
         第二行输入一个长度为 $n$ ，且仅由小写字母构成的字符串 $s$。  
         第三行输入一个长度为 $n$ ，且仅由小写字母构成的字符串 $t$。  
  
 除此之外，保证所有的 $n$ 之和不超过 $10^6$ 。

### 输出描述：

对于每一组测试数据，在一行上输出两个整数，代表最长 $lcp$ 长度和在此条件下最小的 $i$ 。

## 示例1

输入：
```
3
6
baabaa
aabbbb
3
abc
bac
2
ab
cd
```

输出：
```
4 3
3 2
0 1
```


说明：

对于第一组测试数据，我们这样描述整个过程：  
- 选择前缀长度为 1 翻转 $s_{1}'=\text{"}baabaa\text{"}，lcp⁡=0$；  
- 选择前缀长度为 2 翻转 s$s_{2}'=\text{"}ababaa\text{"}~,~lcp⁡=1$ ；  
- 选择前缀长度为 3 翻转 $s_{3}'=\text{"}aabbaa\text{"}，lcp⁡=4$；  
- 选择前缀长度为 4 翻转 $s_{4}'=\text{"}baabaa\text{"}，lcp⁡=0$；  
- 选择前缀长度为 5 翻转 $s_{5}'=\text{"}abaaba\text{"},lcp⁡=1$；  
- 选择前缀长度为 6 翻转 $s_{6}'=\text{"}aabaab\text{"}，lcp⁡=3$；  
 所以最长的公共前缀为 4 ，与此同时最小的翻转下标为 3 。
### 思路：
1. 一个一个的生成 $s_{k}'$ 再去比较 $lcp$ ,复杂度较高，需要一个更快的方法来计算
2. 字符串哈希提供了 $O(1)$ （在 $O(N)$ 预处理后）的方法
3. 对于每个 $k$ ，我们想找到最大的 $L$ ，使得 $s_{k}'[1\dots L]==t[1\dots L]$ ，这个 $L$ 是单调的（如果长度 $L$ 的前缀匹配，那么所有长度小于 $L$ 的前缀也匹配），因此我们可以对 $L$ 进行二分查找
### 步骤：
1. 预处理：
	- 为了处理字符串哈希，我们需要一个质数 $base$ 和一个大质数 $mod$ 
	- 计算字符串 $s$,$t$ 以及 $s$ 的反转串 $reverse(s)$ 的前缀哈希值数组，同时预计算 $base$ 的各次幂
	- 这样我们就可以在 $O(1)$ 的时间内查询任意字串的哈希值
2. 主循环：
	- 遍历所有可能的翻转长度 $k$ 从 1 到 $n$
	- 对于每个 k ，二分查找 $lcp(s_{k}',t)$ 的长度 $L$,范围是 $[0,n]$
	- `check`函数：
		- 该函数判断 $s_{k}'$ 的前 $L$ 个字符是否与
		- 该函数判断  $s_{k}'$  的前 $L$ 个字符是否与$t$ 的前 $L$ 个字符完全相同。
        - 如果 $L\leq k$ : 我们需要比较 $reverse(s[1\dots L])$ 和 $t[1\dots L]$ 。这可以通过比较 `reverse(S)` 的一个子串和  $t$ 的一个前缀的哈希值来完成。
        - 如果 $L>k$ : 我们需要同时满足两个条件：
            1. $revrese(s[1\dots k])==t[1\dots k]$ (第一部分完全匹配)。
            2. $s[k + 1\dots L]==t[k+1\dots L]$ (第二部分的延伸匹配)。 这两个条件都可以通过 $O(1)$ 的哈希比较来验证。
	- 二分查找结束后，我们得到了当前 $k$ 对应的 $LCP$ 长度。我们用它来更新全局的最大 $LCP$ 和对应的最小 $k$。
### code:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

int n;
string s,t;
int z[1010101];

//计算t的前缀匹配长度
void Z() {
    int l = 0,r = 0; //维护当前匹配的最右区间[l,r) : t[l,r - 1]
    for (int i = 1 ; i < n ; ++i) {
        //利用已计算的z值初始化，避免重复比较
        if (i < r) {
            z[i] = min(r - i,z[i - l]);
        }
        else {
            z[i] = 0;
        }
        //扩展匹配长度，更新区间[l,r)
        if (i + z[i] >= r) {
            l = i;
            r = i + z[i];
            while (r < n && t[r] == t[r - l]) {
                ++r;
                z[i]++;
            }
        }
    }
}

void solve() {
    cin >> n >> s >> t;
    Z();
    
    int lcpr = 0,lcpl; //lcpr：未翻转部分的lcp，lcpl：翻转前缀部分的lcp
    int l = n - 1,r = n - 1; //维护翻转前缀匹配的区间指针

    int pos = n; //最优翻转下标（0-based）
    int res = 0; //最大lcp长度

    // 反向遍历i（0-based）：对应题目中翻转前i+1个字符（1-based）
    for (int i = n - 1 ; i > -1 ; --i) {
        // 步骤1：初始化lcpl（翻转前缀的匹配长度）
        if (i > r) {
            lcpl = min(i - r,z[l - i]);
        }
        else {
            lcpl = 0;
        }

        // 步骤2：扩展lcpl的匹配长度
        if (i - lcpl <= r) {
            l = i;
            r = i - lcpl;
            // 翻转前缀的字符对应：s_i'[p] = s[i-p]（p < i+1）
            while (r > -1 && s[r] == t[l - r]) {
                --r;
                lcpl++;
            }
        }

        // 步骤3：计算当前翻转后的总LCP长度
        // - lcpl > i → 翻转前缀（前i+1个）全匹配，需加未翻转部分的lcpr
        // - 否则 → 仅匹配翻转前缀的前lcpl个
        int lcp = (lcpl > i ? lcpl + lcpr : lcpl);

        // 步骤4：更新最大LCP和最小下标（反向遍历保证先遇到更小的i）
        if (lcp >= res) {
            res = lcp;
            pos = i;
        }

        // 步骤5：更新未翻转部分的LCP（lcpr）
        lcpr = (s[i] == t[i] ? lcpr + 1 : 0);
    }

    // 转换为题目要求的1-based下标输出
    cout << res << ' ' << pos + 1 << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
