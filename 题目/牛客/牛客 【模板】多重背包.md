*Algorithm :* [[动态规划]]，[[贪心算法]]
## 描述

对于给定的 $n$ 种物品和一个容量为 $m$ 的背包，每种物品数量为 $s_i$ ，且有体积 $w_i$ 和价值 $v_i$ 两种属性。你可以选取一些物品放入背包带走，求解在装入物品总体积不超过 $m$ 的前提下，能带走的最大物品价值。

> 提示 
> 本题为多重背包模板题，您可以使用二进制优化的多重背包模板通过本题，其实现时间复杂度为 $\displaystyle \mathcal O\left(T \times m \sum_{i=1}^n \log s_i \right)$ ，该复杂度稍大，所以本题的时间限制较为宽松。您也可以使用单调队列优化的多重背包模板通过本题，其实现时间复杂度为 $\mathcal O(T \times nm)$ 。  
>   
> 特殊测试点信息：  
> 	- 测试点 $15$ ，体积均为 $0$ ；  
> 	- 测试点 $16$ ，体积和价值均为 $0$ 。

### 输入描述：

每个测试文件均包含多组测试数据。第一行输入一个整数 $T\left(1\leqq T\leqq 10 \right)$ 代表数据组数，每组测试数据描述如下：  
  
在一行上输入两个整数 $n,m\left( 1 \leqq n,m \leqq 3000 \right)$ 代表物品数量、背包容量。  
此后 $n$ 行，第 $i$ 行输入三个整数 $w_i,v_i,s_i \left( 0 \leqq w_i,v_i \leqq m;\ 1 \leqq s_i \leqq 10^6 \right)$ 代表第 $i$ 件物品的体积、价值、数量。

### 输出描述：

对于每一组测试数据，在一行上输出一个整数，代表能带走的最大物品价值。

## 示例1

输入：
```txt
1
2 5
1 4 5
1 1 4
```

输出：
```txt
20
```
### 思路：
1. 二进制拆分的原理
	- 多重背包的问题是 “每种物品有 $s_i$ 个 ，枚举 $0～ s_i$ 个的时间复杂度太高（$s_i$ 可达 1e6）。
	- 二进制拆分的思路：将 $s_i$ 个物品拆分成若干 “物品组”（每组包含`count`个原物品），这些组可以组合出 $0～s_i$ 之间的**任意数量**。
	- 二进制拆分的目标不是 “单个 cnt 等于 0~num 中的某一个数”，而是：
	- 把 num 拆成若干个「cnt 值」（比如 1、2、2），通过**选 / 不选这些 cnt 值的组合相加**，能得到 0~num 中的**任意一个数**。
	- 举个最容易理解的例子：num=5，拆分出的 cnt 是「1、2、2」。
	- ✅ 不是让 1、2、2 这三个数本身覆盖 0~5，而是**选其中几个相加**覆盖：

|想要凑的数量（0~5）|选哪些 cnt 相加|结果|
|---|---|---|
|0（不选任何物品）|不选任何 cnt|0|
|1|选 1|1|
|2|选第一个 2|2|
|3|选 1+2|3|
|4|选 2+2|4|
|5|选 1+2+2|5|
2. 01 背包的逆序遍历
	- `for(int j=m;j>=w[i]*count;j--)`：逆序遍历容量`j`（从最大容量`m`到当前物品组的体积`w[i]*count`）。
    - 逆序的目的：避免同一物品组被重复选取（如果正序遍历，`dp[j]`更新后会被后续的`dp[j+w[i]]`重复使用，变成完全背包）。
    - 遍历下限`w[i]*count`：容量小于物品组体积时，无法选取该组，无需处理。

### ACcode:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

void solve() {
    int n,m; //n为物品种类，m为背包总容量
    cin >> n >> m;
	
	//w存储每种物品的体积，v存储每种物品的价值，s存储美中物品的数量
    vector<int> w(n + 1),v(n + 1),s(n + 1);

    for (int i = 1 ; i <= n ; ++i) { 
        cin >> w[i] >> v[i] >> s[i];
    }

    vector<ll> dp(m + 1,0); //dp[j]表示背包容量为j时，能装的最大价值

    for (int i = 1 ; i <= n ; ++i) { //遍历每种物品
        int num = s[i]; //当前物品的剩余数量
        
        //二进制拆分：将num拆分成1，2，4，8等2的幂次组合
        for (int k = 1 ; num > 0 ; k <<= 1)  { 
            int cnt = min(k,num); //本次拆分的物品数量（不超过剩余数量）
            num -= cnt;

			//0-1背包的逆序遍历：避免同一物品被重复选取
            for (int j = m ; j >= w[i] * cnt ; --j) {
	            //状态转移：选/不选当前物品组，取价值更大的情况
                dp[j] = max(dp[j],dp[j - w[i] * cnt] + v[i] * cnt);
            }
        }
    }
    cout << dp[m] << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
