描述
--

Alice 和 Bob 在航天基地堵桥，他们提前准备好了一堆全家福，然后准备用这些全家福玩一个游戏，谁输了就会失去浮木。  
  
他们已经随机地把全家福分为了 $n$ 堆，第 $i$ 堆有 $a_i$ 张全家福。两个人轮流进行游戏，每次轮流从全家福堆中取全家福。  
  
每一步，轮到这一步取全家福的人都必须从 **第一个非空全家福堆**（即编号最小且至少有一个全家福的全家福堆）中取走任意正整数张全家福，取走最后一张全家福的人会获胜而保住浮木。

Alice 想知道，如果自己先手，且自己和 Bob 都采取最优策略，最终谁能保全自己的浮木？

### 输入描述：

输入的第一行包含一个整数 $T$（$1\leqq T\leqq 10$），表示测试用例的数量。接下来的 $2T$ 行描述每个测试用例，每个测试用例的格式如下：  
- 第一行输入一个整数 $n$（$1\leqq n\leqq 10^5$），表示全家福堆的数量。  
- 第二行输入 $n$ 个整数 $a_1,\ldots,a_n$（$1\leqq a_i\leqq 10^9$），其中 $a_i$ 表示第 $i$ 堆全家福的数量。  
  
保证所有测试用例中 $n$ 的总和不超过 $10^5$。

### 输出描述：

对于每个测试用例输出一行一个字符串：如果 Alice 会获胜，输出 $\texttt{Alice}$；否则输出 $\texttt{Bob}$。

示例1
---

输入：
```txt
2
6
1 1 4 5 1 4
6
1 1 1 4 4 5
```

输出：
```txt
Alice
Bob
```

## 思路：
1. 游戏的胜负跟先手有很大关系，先从简单的开始：当只有一堆时，先手可以直接取胜；当有两堆时，当第一堆 $x1>1$ 时，先手可以取 $x1-1$ 个，这样后手只能取第一堆剩下的 $1$ 个，但是当第一堆只有 $1$ 个时，先手只能取剩下的 $1$ 个。
2. 那么现在给定一个常规情况，比如3 1 2 1
3. 我们可以反着来，对于先手来说，他一定想取最后一堆（4堆）来获胜，那么3堆的最后一定要由后手来取，这里我们看到3堆 = 2，那么先手可以在轮到3堆的时候取 $x3-1$ 个也就是1个，保证3堆的最后是后手来取，那么2堆的最后一定要是后手来取，观察到 $2堆=1$ ，那么轮到1堆的时候先手就需要把1堆的全部取走。
4. 现在再来一个情况 3 1 1 1 反着来，我们可以得到
	- 1.先手1堆取2个
	- 2.后手只能取1堆剩下1个
	- 3.先手取2堆的1个
	- 4.后手取3推的1个
	- 5.先手取4堆的一个
5. 看到这里我们就知道"先手"肯定能赢，先手选择性的取当前堆的所有或者当前堆的元素个数-1个，从而限制后手。
6. 但是注意这里的先手并不是谁先开始选择谁就是先手，比如对于上面的情况$3 1 2 1$ 我们已经知道先手肯定赢，但是如果是这种情况呢
7. 1 3 1 2 1，先手无论如何只能取第一堆中的1个，从而让出了先手，后手也就成了游戏的主导。
8. 但如果是1 1 3 1 2 1，先手后手都只能取1堆中的个和2堆中的1个，这样先手也就不变了。
9. 由此我们可知对于前面连续的元素个数是1的堆，会影响谁是真正的先手。所以我们需要找到真正的先手，这里可以把特殊情况处理一下，也就是所有堆的元素全为1的情况，之后找到真正的先手，那么真正的先手也就能获胜。
## ACcode:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

void solve() {
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0 ; i < n ; ++i) {
        cin >> a[i];
    }
    int i = 0;
    //计算前面连续1的个数
    while (a[i] == 1 && i < n) {
        ++i;
    }
	
	//特判：如果全是1，那就按顺序一个一个拿
    if (i == n) {
        if (i % 2 == 0) {
            cout << "Bob" << '\n';
        }
        else {
            cout << "Alice" << '\n';
        }
    }
    //如果不全为1，则先手赢
    //如果有奇数个连续1，先手将会被更换为Bob
    //如果为偶数，则先手还是Alice
    else {
        if (i % 2 == 0) {
            cout  << "Alice" << '\n';
        }
        else {
            cout << "Bob" << '\n';
        }
    }
}
int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
