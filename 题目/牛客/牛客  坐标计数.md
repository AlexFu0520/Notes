*Algorithm :* [[位运算]]
## 描述
定义一个坐标变换，坐标 (x,y) 变换后变为 $(~x\oplus y~,~∣x−y∣~)$。  
给定一片矩形区域，计算区域内有多少个整数点在经过有限次变换后变为 (0,0)。  
### 输入描述：
输入第一行一个数字  $t$ , $1≤t≤50$ ，表示测试数据组数  
接下来一行四个数字  $1≤x_{1},y_{1},x_{2},y_{2}≤105$ 代表给出的矩形区域  
$(x_{1},y_{1})$ 为矩形区域左下角， $(x_{2},y_{2})$ 表示矩形右上角，包含边界上的点。  
输入保证有 $x_{1}<x_{2}~,~y_{1}<y_{2}​$ 
### 输出描述：
输出区域内满足变换要求的整数点个数
## 示例1
输入：
```plaintext
2
1 1 3 4
1 2 2 5
```

输出：
```plaintext
12
8
```
### 思路：
1. 通过打表可以发现，无论x，y为何值，都可以通过有限的步数换为 (0 , 0)
2. 打表代码如下：
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

bool check(int x,int y,int k) {
    if (x == 0 && y == 0) return true;
    if (k == 1004) return false;
    return check(x^y,abs(x - y),k + 1);
}

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    for (int i = 0 ; i < 100 ; ++i) {
        for (int j = 0 ;  j < 100 ; ++j) {
            cout << check(i,j,0) << ' ';
        }
        cout << '\n';
    }
    return 0;
}
```
3. 最终仅需计算该矩形面积内点的个数
### code:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

void solve() {
    ll x1,x2,y1,y2;
    cin >> x1 >> y1 >> x2 >> y2;

    cout << (x2 - x1 + 1) * (y2 - y1 + 1) << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);


    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
