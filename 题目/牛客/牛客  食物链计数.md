*Algorithm :* [[图论]]
## 描述

给定一张包含 $n$ 个生物、$m$ 条捕食关系的食物网，可形式化为一张**有向无环图**（DAG）。每条有向边由一对整数 $u,v$ 表示：生物 $v$ 捕食生物 $u$。  
  
在生物学意义上，一条**食物链**被定义为：  
-  起点是**生产者**——不会捕食其它生物（即**出度为 $0$**）；  
-  终点是**顶级消费者**——不会被其它生物捕食（即**入度为 $0$**）；  
-  从生产者到顶级消费者，需要沿捕食方向（即有向边方向）依次连接。  
  
请计算图中这样的食物链共有多少条。题目保证答案不超过 $10^9$。  

### 输入描述：

第一行输入两个整数 $n,m$ $(1\leqq n\leqq10^5,\ 0\leqq m\leqq10^5)$——生物数量与捕食关系数量。  
  
接下来 $m$ 行，每行输入两个整数 $u,v\ (1\leqq u,v\leqq n)$，表示生物 $v$ 捕食生物 $u$（有向边 $u\to v$）。  

### 输出描述：

输出一个整数，表示满足定义的食物链数量。  

## 示例1


输入：
```txt
10 16
1 2
1 4
1 10
2 3
2 5
4 3
4 5
4 8
6 5
7 6
7 9
8 5
9 8
10 6
10 7
10 9
```

输出：
```txt
9
```

说明：所有出度为 $0$ 的顶点为生产者 $\{3,5,8\}$，所有入度为 $0$ 的顶点为顶级消费者 $\{1\}$。沿有向边统计可得共 $9$ 条满足要求的路径。
## 思路：
1. 计算每个点到入度，将入度为0的点加入队列
2. 取出对首，将它指向的所有点的入度减一，如果某点的入度变为0则入队，重复直到队列为空
3. 用拓扑排序+dp
4. 设dp\[u]表示从某个生产者到u到路径数
5. 初始化：所有入度为0且有出边的生产者 dp\[u]=1，入队
6. 按拓扑排序处理：对于当前点u，遍历它的所有出边 u-> v，累加路径数，dp\[v] += dp\[u]，并将v的入度减一，若为0则入队
7. 最后，所有出度为0的点的dp值之和就是答案
### ACcode:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
typedef long long ll;
using namespace std;

struct node {
    int ru;
    int cs = 0; //次数
    vector<int> to;
};

void solve() {
    int n,m;
    cin >> n >> m;
    vector<node> a(n + 1);
    for (int i = 0 ; i < m ; ++i) {
        int u,v;
        cin >> u >> v;
        a[u].to.push_back(v);
        a[v].ru++; //入度+1
    }

    queue<node> q;
    for (int i = 1 ; i <= n ; ++i) {
        //判断是否为可以作为起点的生产者
        if (a[i].ru == 0 && a[i].to.size() != 0) {
            a[i].cs = 1;
            q.push(a[i]); //加入队列
        }
    }

    int ans = 0;
    while (!q.empty()) {
        node now = q.front();
        q.pop();
        for (int i : now.to) {
            a[i].ru--; 
            a[i].cs += now.cs; //计算在i的前面一共有几条食物链
            if (a[i].ru == 0) {
                q.push(a[i]); 
            }
        }
        //判断now是否为顶级消费者，并且它的所有食物链都计算过了（入度变为0）
        if (now.to.empty() && now.ru == 0) ans += now.cs;
    }
    cout << ans << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
