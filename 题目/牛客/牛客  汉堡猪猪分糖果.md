*Algorithm :* [[位运算]]
描述
--

汉堡猪猪有 $n$ 颗糖果，准备分给 $m$ 位小朋友，且要求**每位小朋友至少获得一颗糖果**。设最后第 $j$ 位小朋友得到的糖果数为 $c_j$，则有  
  
$$\displaystyle c_1+c_2+\dots+c_m = n,\quad c_j \geqq 1$$
  
汉堡猪猪关注分配结果的**按位与值**：  
$$\displaystyle c_1\ \&\ c_2\ \&\ \dots\ \&\ c_m$$

并想让该值尽可能大。请你帮他计算在最优分配下该按位与的最大可能值。  

### 输入描述：

每个测试文件包含多组测试数据。第一行输入整数 $T\left(1 \leqq T \leqq 10^5\right)$ 表示数据组数。  
随后 $T$ 行，每行输入两个整数 $n,m\left(1 \leqq m \leqq n \leqq 10^9\right)$ 表示糖果总数与小朋友人数。  

### 输出描述：

对每组测试数据，输出一行，表示在满足 $c_j \geqq 1$ 的前提下，$c_1\ \&\ c_2\ \&\ \dots\ \&\ c_m$ 的最大可能值。  

示例1
---

输入：
```txt
4
8 3
3 2
5 1
514114 114514
```

输出：
```txt
2
0
5
4
```
## 思路：
- 首先我们知道，根据**位与&** 的特性，就算有某些c_i 占据了高的数位，**有一个c_j 在该位上为0，那&起来的结果肯定为0**。

![](https://uploadfiles.nowcoder.com/images/20260216/288628362_1771225705686/D2B5CA33BD970F64A6301FA75AE2EB22)

- 所以我们希望位与和的结果足够大，就得**保证每个c_i 的最高位都为1**。
- 这**启发我们从高位到低位枚举贪心**，看看n 能分成m个的哪些位。
## ACcode:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

void solve() {
    int n,m;
    cin >> n >> m;
    ll tmp = n,ans = 0;
    for (int i = 31 ; i >= 0 ; --i) {
        ll bit = (1ll << i);
        //对于当前位 bit = 2^i，若能给每个小朋友都分配 bit 颗糖（共需 bit * m 颗），则这一位可以保留（所有 c_j 的这一位都是 1）
        if (tmp >= bit * m) {
            tmp -= bit * m;
            ans |= bit;
        }
        //如果糖果大于填满低于第i位第所有位的数量，则多出来的糖直接去掉不发了
        //如果非要发的话，第i位就不可能变为1了
        else if (tmp > m * (bit - 1)) {
            tmp -= (tmp - bit  * m + m + bit - 1) / bit  * bit;
        }
    }    
    cout << ans << '\n';
}

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
