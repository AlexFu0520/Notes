*Algorithm :* [[二分算法]]，[[滑动窗口与双指针]]

描述
--

$dd$作为集训队的队长，一直掌管着集训室的空调遥控器，她需要调整温度使队员们更好地进入训练状态，已知集训室一共有$n$名队员，每位队员都有一个温度诉求$a[i](1≤i≤n)$，当室内温度为$K$时，当且仅当$|a[i]-K|≤p$时，这个队员能够正常进入训练状态，否则就会开始躁动，作为队长，$dd$需要调整好温度，她想知道，在最佳情况下，最多有多少队员同时进入训练状态

### 输入描述：

第一行两个数n,p(1≤n,p≤1000000),含义如题面描述  
接下来一行n个数a\[i\](1≤a\[i\]≤1000000)表示每个队员的温度诉求  

### 输出描述：

输出一个数字，表示最多有多少队员同时进入训练状态  

示例1
---

输入：
```txt
6 2
1 5 3 2 4 6
```

输出：
```txt
5
```

说明：
```txt
温度调成3或4，都可以满足5名队员同时进入训练状态
```
## 解法1 思路：
1. 排序+二分
2. $\left | a_{i} - k\right | \leq p$  => $k-p\leq a_{i}\leq k+p$ 
3. 枚举温度时，只需找到区间 $[k-p,k+p]$ 内的数即可，也就是找到数 $k-p$ 的左端点，$k+p$ 的右端点
## ACcode (手写二分):
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void solve() {
    int n,p;
    cin >> n >> p;
    
    vector<int> a(n);
    int mx = 0,mi = 0;
    for (int i = 0 ; i < n ; ++i) {
        cin >> a[i];
        if (a[i] > mx) mx = a[i];
        if (a[i] < mi) mi = a[i];
    }
    sort(a.begin(),a.end());

    int ans = 0;
    for (int k = mi ; k <= mx ; ++k) {
        int x = k - p,y = k + p;
        int l = 0,r = n - 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (a[mid] >= x) r = mid - 1;
            else {
                l = mid + 1;
            }
        }
        int begin = l;
        l = 0,r = n - 1;
        while (l <= r) {
            int mid = l + (r - l) / 2;
            if (a[mid] > y) r = mid - 1;
            else {
                l = mid + 1;
            }
        }
        int end = l;
        ans = max(ans,end - begin);
    }
    cout << ans << '\n';
}

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```

## 滑动窗口解法思路：
1.  $\left | a_{i} - k\right | \leq p$  => $k-p\leq a_{i}\leq k+p$ 还是这个关系
2. 要先对数组排序
3. 对数组进行滑动窗口，保证区间最大值 - 最小值 <= 2 * p
## ACcode:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

void solve() {
    int n,p;
    cin >> n >> p;
    
    vector<int> a(n);
    int mx = 0,mi = 0;
    for (int i = 0 ; i < n ; ++i) {
        cin >> a[i];
        if (a[i] > mx) mx = a[i];
        if (a[i] < mi) mi = a[i];
    }
    sort(a.begin(),a.end());

    int ans = 0;
    int left = 0;
    for (int right = 0 ; right < n ; ++right) {
        while (a[right] - a[left] > 2 * p) ++left;
        ans = max(ans,right - left + 1);
    }
    cout << ans << '\n';
}

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
