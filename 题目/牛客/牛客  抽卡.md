描述
--

王子连接的国服终于上线啦~  
已知王子连接的抽卡系统如下：  
共有 $n$  个卡池，第 $i$ 个卡池共有$a_i$  种卡，每张卡的出货率都是相等的（也就是说该卡池单次抽卡，每种卡出货率是 $1/a_i$ ）。  
第 $i$  个卡池中，你有 $b_i$  种卡是自己很想要的。  
现在的问题是，如果每个卡池里都单抽一次，能抽到自己想要的卡的概率是多少？  
可以证明，这个概率一定可以写成 $a/b$  形式的分数。最后输出该分数在模 $10^9+7$意义下的值就可以了。  
即输出满足 $b*x\%1000000007=a$ 的最小非负整数 $x$ 。

### 输入描述：

第一行输入一个正整数 $n（1≤n≤100000）$ 。  

第二行输入 $n$  个正整数 $a_i$。  

第三行输入 $n$  个正整数 $b_i$ ，代表第 $i$  个卡池的你想要的卡种类数量。$（1≤b_i≤a_i≤100000）$ 

### 输出描述：

一个整数，表示该概率在模 $10^9+7$ 意义下的值。

示例1
---

输入：
```txt
2
3 4
1 1
```

输出：
```txt
500000004
```

说明：
```txt
能抽到自己想要的卡的概率是1/2，由于2*500000004%1000000007=1，故输出500000004。
```
## 思路：
全没抽到的概率：$$P_{fail}=\prod_{i=0}^{n-1} \frac{a_{i}-b_{i}}{a_{i}}=\frac{\prod_{i=0}^{n-1}a_{i}-b_{i}}{\prod_{i=0}^na_{i}}$$
令$mol=\prod_{i=0}^{n-1}a_{i}-b_{i},den={\prod_{i=0}^{n-1}a_{i}}$

至少抽中一张的概率：$$P=1-P_{fail}=1- \frac{mol}{den}= \frac{den-mol}{den}$$
在模运算下：$$\frac{den-mol}{den}在模运算下=(den-mol)*den^{-1} mod~MOD$$
其中 $den^{-1}$ 时 $den$ 的逆元
## ACcode:
```cpp
#include<iostream>
#include<algorithm>
#include<vector>
#define inf 0x3f3f3f3f
typedef long long ll;
using namespace std;

const int mod = 1e9 + 7;

ll qpow(ll base,ll power,ll mod) {
    ll ans = 1;
    while (power) {
        if (power & 1) {
            ans = (ans * base) % mod;
        }
        power >>= 1;
        base = (base * base) % mod;
    }
    return ans;
}

void solve() {
    int n;
    cin >> n;
    vector<int> a(n),b(n);
    for (int i = 0 ; i < n ; ++i) cin >> a[i];
    for (int i = 0 ; i < n ; ++i) cin >> b[i];

    ll mol = 1,den = 1;
    for (int i = 0 ; i < n ; ++i) {
        int diff = (a[i] - b[i] + mod) % mod;
        mol = mol * diff % mod;
        den = den * a[i] % mod;
    }

    ll ans;
    ans = (den - mol + mod) % mod * qpow(den,mod - 2,mod) % mod;
    cout << ans << '\n';
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);

    int T = 1;
    // cin >> T;

    while (T--) {
        solve();
    }
    return 0;
}
```
