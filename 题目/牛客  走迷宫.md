*Algorithm :* [[图论]]
## 描述

给定一个 $n×m$ 的网格。你从起点 $(x_{s},y_{s})$ 出发，每一次可以向上、下、左、右移动一步（若不超出边界）。某些格子上存在障碍物，无法经过。求从 $(x_{s},y_{s})$ 移动到终点 $(x_{t},y_{t})$ 的最少步数；若无法到达，输出 −1。  
### 输入描述：

在一行上输入两个整数 $n,m(1≦n,m≦1000)$ ，代表网格的行数与列数。  
在一行上输入四个整数 $x_{s},y_{s},x_{t},y_{t}(1\leq x_{s},x_{t}\leq n ; 1\leq y_{s},y_{t}\leq m)$ ，代表起点与终点的坐标。  
此后 $n$ 行，第 $i$ 行输入一个长度为 $m$ 的字符串 $g_{i}$​，其中  
∙ ∙若 $g_{i,j}='*'$，表示第 $i$ 行第 $j$ 列为障碍物；  
∙ ∙若 $g_{i,j}='.'$ ，表示该格子可通行。  
保证起点所在格子可通行。  
### 输出描述：

输出一个整数，表示最少移动次数；若无法到达，输出 −1。  
## 示例1

输入：
```plaintext
5 5
1 1 5 5
.....
****.
.....
**.**
.....
```

输出：
```plaintext
12
```

### 思路：
1. 建图
2. BFS，使用dis数组记录从起点到每个点到最短路程
### code:
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
typedef long long ll;
using namespace std;

vector<pair<int,int>> dirs = {{0,1},{0,-1},{1,0},{-1,0}};

void solve() {
    int n,m;
    cin >> n >> m;

    int sx,sy,tx,ty;
    cin >> sx >> sy >> tx >> ty;
    sx--,sy--,tx--,ty--;

    vector<vector<char>> grid(n,vector<char>(m));
    for (int i = 0 ; i < n ; ++i) {
        string s;
        cin >> s;
        for (int j = 0 ; j < s.size() ; ++j) {
            grid[i][j] = s[j];
        }
    }
	
	//特判：如果终点障碍物
    if (grid[tx][ty] == '*') {
        cout << -1  << '\n';
        return;
    }
    
	//如果终点与起点相同
    if (sx == tx && sy == ty) {
        cout << 0 << '\n';
        return;
    }

    vector<vector<int>> dis(n,vector<int>(m,-1));
    queue<pair<int,int>> q;
    q.push({sx,sy});
    dis[sx][sy] = 0;

    while (!q.empty()) {
        auto [x,y] = q.front();
        q.pop();

        for (auto [dx,dy] : dirs) {
            int nx = x + dx,ny = y + dy;
            //dis[nx][ny] == -1 防止出现重复走的情况
            if (nx >= 0 && nx < n && ny >= 0 && ny < m && grid[nx][ny] == '.' && dis[nx][ny] == -1) {
                dis[nx][ny] = dis[x][y] + 1;
                if (nx == tx && ny == ty) { //走到终点
                    cout << dis[nx][ny] << '\n';
                    return;
                }
                q.push({nx,ny});
            }
        }
    }
    cout << -1 << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
