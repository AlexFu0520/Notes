*Algorithm :* [[图论]] 
## 描述
牛牛得到了一堆神奇的恶魔果实，每个恶魔果实都给了牛牛一个改变数字的能力，可以把数字 $a$ 变成数字 $b$ ，现在牛牛有一个数字 $x$ ，他想知道吃完这 $n$ 个恶魔果实后，他可以把数字 $x$ 变成多少种的数。

**注：每一个恶魔果实的能力可以重复使用多次，当然也可以不用，存在相同能力的恶魔果实.**
### 输入描述：
第一行输入一个正整数 $x(1≤x≤10^9)$ ，和一个正整数 $n(1≤n≤10^6)$ ,接下来 $n$ 行，每行一个 $a(0≤a≤9 )$ 和一个 $b(1≤b≤9)$ 。
### 输出描述：
表示可以变的数字种类数，答案对104+7取模
## 示例1

输入：
```plaintext
456 3
5 6
4 5
4 5
```

输出：
```plaintext
6
```

说明：
```plaintext
456 556 656 466 566 666
```

## 示例2

输入：
```plaintext
111 1
1 2
```

输出：
```plaintext
8
```

说明：
```plaintext
111 112 121 122 211 212 221 222
```

### 思路：
##### 我们只需知道，对于每个数字 0-9 ，它最终能变成哪些数字（包括它自己），然后把 x 的每一位的可能性乘起来，就得到总数了
##### 用 to\[a]\[b] 表示是否有能力把 a 变成 b ，输入 n 条规则 ，就把对应的 to\[a]\[b] 设为true

#### 传递闭包（Floyd-Warshall）
1. 因为能力可以重复使用，所以如果 a 能变成 c ，c 能变成 b ，那么 a 也能变成 b 
2. 我们使用三重循环
```cpp
for (int k = 0 ; k < 10 ; ++k) { // k 是中间人
	for (int i = 0 ; i < 10 ; ++i) { // i想交朋友
		for (int j = 0 ; j < 10 ; ++j) { //j 是目标
			if (to[i][k] && to[k][j]) { //如果i 认识k，k也认识j
				to[i][j] = true; //那么i也认识j
			}
		}
	}
}
```
3. 这样循环完，to\[ i ]\[ j ] 就表示 i 能不能通过任意次变换变为 j 
4. 每个数字当然可以保持不变，所以把 to\[ i ]\[ i ] 也设为true
5. 对于每个 i ，数一数 to\[ i ]\[ j ] 为 true 的 j 有多少个，存到 shu\[ i ]里
6. 把 x 的每一位都拆出来，比如 x = 123 ，就分别取 3，2，1
7. 答案 = shu\[3] * shu\[2] * shu\[1] % mod
### code
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
typedef long long ll;

ll MOD = 1e4 + 7;

void solve() {
    ll x,n;
    cin >> x >> n;
    vector<vector<bool>> to(10,vector<bool> (10,false));
    for (int i = 0 ; i < n ; ++i) {
        int a,b;
        cin >> a >> b;
        to[a][b] = true;
    }

    for (int k = 0 ; k < 10 ; ++k) {
        for (int i = 0 ; i < 10 ; ++i) {
            for (int j = 0 ; j < 10 ; ++j) {
                if (to[i][k] && to[k][j]) {
                    to[i][j] = true;
                }
            }
        }
    }

    for (int i = 0 ; i < 10 ; ++i) to[i][i] = true;

    vector<int> shu(10);
    for (int i = 0 ; i < 10 ; ++i) {
        int cnt = 0;
        for (int j = 0 ; j < 10 ; ++j) {
            if (to[i][j]) ++cnt;
        }
        shu[i] = cnt;
    }

    ll ans = 1;
    while (x) {
        ans = ans * shu[x % 10] % MOD;
        x /= 10;
    }
    cout << ans << '\n';
}               

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);


    int T = 1;
    // cin >> T;
    while (T--) {
        solve();   
    }
    return 0;
}
```
