# 一、线性插值（Linear Interpolation）

## 1. 核心原理

线性插值是最简单的插值方法，其核心思想是：**在两个相邻的已知数据点之间，用一条直线来近似模拟数据的变化**。

假设我们有两个已知点 $(x_{0},y_0​)$ 和 $(x_{1}​,y_{1}​)$，现在要估算任意 $x\in[x_{0}​,x_{1}]$ 对应的 y 值，公式为：

$y=y_{0}+(y_{1}−y_{0}​)⋅ \frac{x​−x_{0}}{x_{1}-x_{0}}​​​$

## 2. 代码中的实现

```python
if interp_method == 'linear':
    interp_model = interp1d(
        t_unique, c_unique,          # 已知的时间点和单车数量
        kind='linear',                # 指定线性插值
        fill_value="extrapolate",  # 允许外推（超出已知时间范围也能算）
        bounds_error=False         # 不报错，即使目标点超出范围
    )
```

- **`t_unique`**：清洗后的已知时间点（已排序去重）。
- **`c_unique`**：对应的单车数量。
- **`fill_value="extrapolate"`**：关键参数！如果目标时间点（如 7:00）早于最早的观测时间，或晚于最晚的观测时间，它会按照**最后一段直线的趋势**向外延伸计算。

## 3. 优缺点

|优点|缺点|
|---|---|
|计算简单、速度极快|不够平滑，在数据点连接处可能有 “折角”|
|不会出现剧烈震荡（鲁棒性好）|对于非线性变化的数据，拟合精度有限|
|外推时比较稳定（线性趋势）|-|

# 二、三次样条插值（Cubic Spline Interpolation）

## 1. 核心原理

三次样条是一种更高级的平滑插值方法。它的核心思想是：**将整个数据区间分成若干小段，每一小段用一个三次多项式来拟合，并且保证在连接点处，函数值、一阶导数、二阶导数都是连续的**。

这就意味着，最终画出来的曲线不仅经过所有已知点，而且是 “光滑” 的，没有突兀的拐角。

## 2. 代码中的实现

```python
else:
    # 三次样条插值
    interp_model = CubicSpline(
        t_unique, c_unique,          # 已知数据
        extrapolate=True              # 允许外推
    )
```

- **`CubicSpline`**：SciPy 专门用于三次样条插值的类。
- **`extrapolate=True`**：同样允许对已知时间范围之外的点进行预测。

## 3. 优缺点

|优点|缺点|
|---|---|
|曲线非常光滑，符合现实世界的连续变化规律|计算量比线性插值大（但对于这个数据量可忽略）|
|对非线性趋势的拟合效果更好|如果数据点过少或噪声大，可能会产生 “龙格现象”（剧烈震荡）|
|能很好地反映数据的曲率变化|外推时的不确定性比线性插值大|

# 三、代码中针对拟合的特殊处理

这段代码不仅做了插值，还针对 “共享单车数量” 这个具体场景做了**物理意义修正**：

1. **非负约束**：    
```python
c_pred = np.clip(c_pred, 0, None)
```

`插值可能算出负数（比如外推时），但单车数量不能为负，所以强制最小为 0。`

2. **整数修正**：

```
c_pred = np.round(c_pred).astype(int)
```
`单车数量必须是整数，所以对结果进行四舍五入`
3. **数据清洗（排序去重）**：

```
sorted_idx = np.argsort(t_obs)
t_unique, unique_idx = np.unique(t_sorted, return_inverse=True)
```    

插值算法要求输入数据必须是**按时间递增且无重复**的。如果有同一时间点的多条记录，代码取了平均值。
