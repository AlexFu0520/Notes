# 问题1

 变量名称：标称次品率：$P_{0}$  = 0.1  ，设 $P$ 为样本检测的实际次品率 (样本次品率)
 假设检验：零假设  $H_{{0}}$ ：$P = P_{0}$  备择假设  $H_{1}$ ： $P \neq P_{0}$  , 分别对置信区间为95%和90%进行假设检验
## 方法一：二项分布估计

1. 设次品数为X，X服从于二项分布     $X～B(n,p_{0})$  n就是我们要抽取的样本数量
2. 由于是对次品率进行置信假设检验，所以要将 X 的分布，转化为 P 的分布
3. 根据置信区间估算出要抽的样本数量

### 公式推导

转为P的分布，P也属于二项分布
$$E(P) = E(\frac{X}{n} )=\frac{nP_{0}}{n} = P_{0}  ， D(P) = D(\frac{X}{n} )=\frac{nP_{0} (1-P_{0} )}{n^{2} } = \frac{P_{0}(1 - P_{0} )}{n}$$
二项分布在大样本下可以近似为正态分布，通常可用于估算二项分布的样本量，所以
$$P ～ B(P_{0} ,\frac{P_{0}(1 - P_{0}) }{n} )$$
因为抽样数量不为负数，所以应该取单侧置信区间计算上限
在实际需求中，还需考虑到误差的存在，如果没有误差的话则不能计算抽样样本量
设绝对误差 E=5% 
$$设 Z=\frac{P - P_{0} }{\sqrt{ \frac{P_{0} (1-P_{0}) }{n} }}，则Z～N(0,1)$$
$$P(P-P_{0}\leq E )=1-\alpha $$
$$P(Z * \sqrt{ \frac{P_{0} (1-P_{0}) }{n}} \leq E) =1-\alpha$$
$$P(Z\leq \frac{E}{\sqrt{\frac{ P_{0} (1-P_{0})}{n} }}) = 1-\alpha$$
$$\frac{E}{\sqrt{\frac{ P_{0} (1-P_{0})}{n} }} = Z_{\alpha}$$
$$n = \frac{Z_{\alpha}^{2} P_{0} (1-P_{p})}{E^{2}}$$
根据此公式可以计算出估算的抽样的数量

python代码如下：

```python
import scipy.stats as stats
import math 
#标称次品率
p0 = 0.1
#显著性水平
alpha1 = 0.05
alpha2 = 0.1
#绝对误差
E = 0.05

#计算上alpha分位点
z1 = stats.norm.ppf(1 - alpha1)
z2 = stats.norm.ppf(1 - alpha2)

#根据公式计算最小抽取数目
n1 = math.ceil((z1**2 * p0 * (1 - p0)) / (E**2))
n2 = math.ceil((z2**2 * p0 * (1 - p0)) / (E**2))

print("置信度为95%时，最小抽取数目为：",n1)
print("置信度为90%时，最小抽取数目为：",n2)
```

## 方法二：更准确的二项分布估计

当原假设 $H_{0}$（真实次品率 = $P_{0}$​ ）为真时： $P^​∼B(P_{0}​,\frac{P_{0}​(1−P_{0}​)} {n}​)$

当备择假设 $H_{1}$（真实次品率 = $P_{1}$  ​，且 $P_{1}$ ​> $P_{0}$ ​）为真时：$P^​∼B(P_{1}​,\frac{P_{1}(1−P_{1}​)}{n}​)$

我们要控制第一类错误的概率和第二类错误的概率取相同的一个临界值 $P_{c}$
1. 控制第一类错误，得到临界值 $P_{c}$
$$\frac{P_{c} - P_{0}}{\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }} = Z_{{\alpha}}$$
$$P_{c} =P_{0} +Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }$$
2. 控制第二类错误，得到临界值  $P_{c}$
	检验功效为 $1-\beta$ ，示例代码里为 $\beta=0.2$ 
	取 “刚好接受H0​” 的临界条件，标准化后统计量等于−Zβ​（因为 $P_{1}>P_{0}$ ，$P_{c} <P_{1}$ ​，所以分子为负）：
$$\frac{P_{c} - P_{1}}{\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }} = -Z_{\beta}$$
$$P_{c} =P_{1} -Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }$$
3. 联立临界值 $P_{c}$ ，解出样本量n
$$P_{0} +Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }=P_{1} -Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }$$
$$Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }+Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }=P_{1}-P_{0} $$
$$\sqrt{ \frac{1}{n}}\cdot \left [   Z_{{\alpha}}\sqrt{ P_{0}(1 - P_{0}) }+Z_{\beta}\sqrt{P_{1}(1 - P_{1}) } \right ]==P_{1}-P_{0}$$

$$n= \frac{\left [   Z_{\alpha}\sqrt{ P_{0}(1 - P_{0}) }+Z_{\beta}\sqrt{P_{1}(1 - P_{1}) } \right ]^2}{(P_{1}-P_{0})^2} $$
 python代码如下:
 
```python
import scipy.stats as stats
from math import *

p0 = 0.1
p1 = 0.15

alpha1 = 0.05
alpha2 = 0.1
beta = 0.2

z1 = stats.norm.ppf(1 - alpha1)
z2 = stats.norm.ppf(1 - alpha2)
z3 = stats.norm.ppf(1 - beta)

n1 = ceil((z1 * sqrt(p0 * (1-p0) ) + z3 * sqrt(p1 * (1-p1) ))**2/(p1 - p0) ** 2)
n2 = ceil((z2 * sqrt(p0 * (1-p0) ) + z3 * sqrt(p1 * (1-p1) ))**2/(p1 - p0) ** 2)

print("置信度为95%时，最小抽取数目为：",n1)
print("置信度为90%时，最小抽取数目为：",n2)
```
# 问题二

根据题目，需要编写4个函数，分别处理以下四个问题：
1. 零配件1是否需要检测
2. 零配件2是否需要检测
3. 成品是否需要检测
4. 不合格成品是否需要拆解
每个问题都对应2种结果，总共有16种结果可以直接进行枚举
我们应写一个函数，计算出每种情况的最大期望利润，得出最优决策组合

python代码如下：

```python
# p1_d 零配件1的次品率 , c1_b 零配件1的购买单价 , c1_t 零配件1的检测成本
# p2_d 零配件2的次品率 , c2_b 零配件2的购买单价 , c2_t 零配件2的检测成本 
# final_d 成品次品率 , c_a 装配成本 , c_t 成品检测成本 , p 市场售价
# l_change 调换损失 , c_d 拆解费用 
def cal_profits(p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d):
	max_profit = 0
	best_decision = dict()
	
	#枚举16种情况 ，t1，t2，t，d表示分别表示以上的四个问题
	for t1 in [True,False]:
		for t2 in [True,False]:
			for t in [True,False]:
				for d in [True,False]:
					#计算各个环节的合格概率
					p1_q = 1 if t1 else (1 - p1_d) #零件1合格概率
					p2_q = 1 if t2 else (1 - p2_d) #零件2合格概率
					f_q = p1_q * p2_q * (1 - final_d) #成品合格概率
					f_uq = 1 - f_q #成品不合格概率
					
                    #计算各个环节的成本
					buy = c1_b + c2_b #固定购买成本
					c1 = (t1 * c1_t) + (t2 * c2_t) #零件的总检测成本
					c2 = c_a #固定装配成本
					c3 = t * c_t #成品检测成本
					c4 = 0 if t else f_uq * l_change #调换成本
					c5 = d * f_uq * c_d #拆解成本
					
                    #各个利润
					p1 = p - buy - c1 - c2 - c3 - c4 -c5 #首次生产的总利润
					p2 = p - c1 - c2 - c3 - c4 - c5 #拆解后再生产的总利润
					
                    #拆解循环的概率
					f_c = f_uq if d else 0
					
                    #最终的总利润
					final_profit = p1 + (p2 * f_c) / (1 - f_c) if f_c else 0
					
                    #选取最优的决策组合
					if (final_profit > max_profit):
						max_profit = final_profit
						best_decision = {
							"零配件1是否检测":"是" if t1 else "否",
							"零配件2是否检测":"是" if t2 else "否",
							"成品是否检测":"是" if t else "否",
							"不合格成品是否拆解":"是" if d else "否"
						}
	return best_decision

#对应所给的6种不同的情况
scenario = [
	[0.1,4,2,0.1,18,3,0.1,6,3,56,6,5],
	[0.2,4,2,0.2,18,3,0.2,6,3,56,6,5],
	[0.1,4,2,0.1,18,3,0.1,6,3,56,30,5],
	[0.2,4,1,0.2,18,1,0.2,6,2,56,30,5],
	[0.1,4,8,0.2,18,1,0.1,6,2,56,10,5],
	[0.05,4,1,0.05,18,3,0.05,6,3,56,10,40]
]

cnt = 1
for [p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d] in scenario:
    print(f"=====第{cnt}种情况=====")
    print("最优决策组合为：")
    best_decision = cal_profits(p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d)
    for key,value in best_decision.items():
        print(f"{key}:{value}")
    print('-'*30)
    cnt += 1
```
