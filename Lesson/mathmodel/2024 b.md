# 问题1

 变量名称：标称次品率：$P_{0}$  = 0.1  ，设 $P$ 为样本检测的实际次品率 (样本次品率)
 假设检验：零假设  $H_{{0}}$ ：$P = P_{0}$  备择假设  $H_{1}$ ： $P \neq P_{0}$  , 分别对置信区间为95%和90%进行假设检验
## 方法一：二项分布估计

1. 设次品数为X，X服从于二项分布     $X～B(n,p_{0})$  n就是我们要抽取的样本数量
2. 由于是对次品率进行置信假设检验，所以要将 X 的分布，转化为 P 的分布
3. 根据置信区间估算出要抽的样本数量

### 公式推导

转为P的分布，P也属于二项分布
$$E(P) = E(\frac{X}{n} )=\frac{nP_{0}}{n} = P_{0}  ， D(P) = D(\frac{X}{n} )=\frac{nP_{0} (1-P_{0} )}{n^{2} } = \frac{P_{0}(1 - P_{0} )}{n}$$
二项分布在大样本下可以近似为正态分布，通常可用于估算二项分布的样本量，所以
$$P ～ B(P_{0} ,\frac{P_{0}(1 - P_{0}) }{n} )$$
因为抽样数量不为负数，所以应该取单侧置信区间计算上限
在实际需求中，还需考虑到误差的存在，如果没有误差的话则不能计算抽样样本量
设绝对误差 E=5% 
$$设 Z=\frac{P - P_{0} }{\sqrt{ \frac{P_{0} (1-P_{0}) }{n} }}，则Z～N(0,1)$$
$$P(P-P_{0}\leq E )=1-\alpha $$
$$P(Z * \sqrt{ \frac{P_{0} (1-P_{0}) }{n}} \leq E) =1-\alpha$$
$$P(Z\leq \frac{E}{\sqrt{\frac{ P_{0} (1-P_{0})}{n} }}) = 1-\alpha$$
$$\frac{E}{\sqrt{\frac{ P_{0} (1-P_{0})}{n} }} = Z_{\alpha}$$
$$n = \frac{Z_{\alpha}^{2} P_{0} (1-P_{p})}{E^{2}}$$
根据此公式可以计算出估算的抽样的数量

python代码如下：

```python
import scipy.stats as stats
import math 
#标称次品率
p0 = 0.1
#显著性水平
alpha1 = 0.05
alpha2 = 0.1
#绝对误差
E = 0.05

#计算上alpha分位点
z1 = stats.norm.ppf(1 - alpha1)
z2 = stats.norm.ppf(1 - alpha2)

#根据公式计算最小抽取数目
n1 = math.ceil((z1**2 * p0 * (1 - p0)) / (E**2))
n2 = math.ceil((z2**2 * p0 * (1 - p0)) / (E**2))

print("置信度为95%时，最小抽取数目为：",n1)
print("置信度为90%时，最小抽取数目为：",n2)
```

## 方法二：更准确的二项分布估计

当原假设 $H_{0}$（真实次品率 = $P_{0}$​ ）为真时： $P^​∼B(P_{0}​,\frac{P_{0}​(1−P_{0}​)} {n}​)$

当备择假设 $H_{1}$（真实次品率 = $P_{1}$  ​，且 $P_{1}$ ​> $P_{0}$ ​）为真时：$P^​∼B(P_{1}​,\frac{P_{1}(1−P_{1}​)}{n}​)$

我们要控制第一类错误的概率和第二类错误的概率取相同的一个临界值 $P_{c}$
1. 控制第一类错误，得到临界值 $P_{c}$
$$\frac{P_{c} - P_{0}}{\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }} = Z_{{\alpha}}$$
$$P_{c} =P_{0} +Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }$$
2. 控制第二类错误，得到临界值  $P_{c}$
	检验功效为 $1-\beta$ ，示例代码里为 $\beta=0.2$ 
	取 “刚好接受H0​” 的临界条件，标准化后统计量等于−Zβ​（因为 $P_{1}>P_{0}$ ，$P_{c} <P_{1}$ ​，所以分子为负）：
$$\frac{P_{c} - P_{1}}{\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }} = -Z_{\beta}$$
$$P_{c} =P_{1} -Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }$$
3. 联立临界值 $P_{c}$ ，解出样本量n
$$P_{0} +Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }=P_{1} -Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }$$
$$Z_{{\alpha}}\sqrt{ \frac{P_{0}(1 - P_{0})}{n} }+Z_{\beta}\sqrt{ \frac{P_{1}(1 - P_{1})}{n} }=P_{1}-P_{0} $$
$$\sqrt{ \frac{1}{n}}\cdot \left [   Z_{{\alpha}}\sqrt{ P_{0}(1 - P_{0}) }+Z_{\beta}\sqrt{P_{1}(1 - P_{1}) } \right ]==P_{1}-P_{0}$$

$$n= \frac{\left [   Z_{\alpha}\sqrt{ P_{0}(1 - P_{0}) }+Z_{\beta}\sqrt{P_{1}(1 - P_{1}) } \right ]^2}{(P_{1}-P_{0})^2} $$
 python代码如下:
 
```python
import scipy.stats as stats
from math import *

p0 = 0.1
p1 = 0.15

alpha1 = 0.05
alpha2 = 0.1
beta = 0.2

z1 = stats.norm.ppf(1 - alpha1)
z2 = stats.norm.ppf(1 - alpha2)
z3 = stats.norm.ppf(1 - beta)

n1 = ceil((z1 * sqrt(p0 * (1-p0) ) + z3 * sqrt(p1 * (1-p1) ))**2/(p1 - p0) ** 2)
n2 = ceil((z2 * sqrt(p0 * (1-p0) ) + z3 * sqrt(p1 * (1-p1) ))**2/(p1 - p0) ** 2)

print("置信度为95%时，最小抽取数目为：",n1)
print("置信度为90%时，最小抽取数目为：",n2)
```
# 问题二

根据题目，需要编写函数，分别处理以下四个问题：
1. 零配件1是否需要检测
2. 零配件2是否需要检测
3. 成品是否需要检测
4. 不合格成品是否需要拆解
### 方法：按照总体最大利润去选出最优的决策组合

每个问题都对应2种结果，总共有16种结果可以直接进行枚举
我们应写一个函数，计算出每种情况的最大期望利润，得出最优决策组合
python代码如下：

```python
# p1_d 零配件1的次品率 , c1_b 零配件1的购买单价 , c1_t 零配件1的检测成本
# p2_d 零配件2的次品率 , c2_b 零配件2的购买单价 , c2_t 零配件2的检测成本 
# final_d 成品次品率 , c_a 装配成本 , c_t 成品检测成本 , p 市场售价
# l_change 调换损失 , c_d 拆解费用 
def cal_profits(p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d):
	max_profit = 0
	best_decision = dict()
	
	#枚举16种情况 ，t1，t2，t，d表示分别表示以上的四个问题
	for t1 in [True,False]:
		for t2 in [True,False]:
			for t in [True,False]:
				for d in [True,False]:
					#计算各个环节的合格概率
					p1_q = 1 if t1 else (1 - p1_d) #零件1合格概率
					p2_q = 1 if t2 else (1 - p2_d) #零件2合格概率
					f_q = p1_q * p2_q * (1 - final_d) #成品合格概率
					f_uq = 1 - f_q #成品不合格概率
					
                    #计算各个环节的成本
					buy = c1_b + c2_b #固定购买成本
					c1 = (t1 * c1_t) + (t2 * c2_t) #零件的总检测成本
					c2 = c_a #固定装配成本
					c3 = t * c_t #成品检测成本
					c4 = 0 if t else f_uq * l_change #调换成本
					c5 = d * f_uq * c_d #拆解成本
					
                    #各个利润
					p1 = p - buy - c1 - c2 - c3 - c4 -c5 #首次生产的总利润
					p2 = p - c1 - c2 - c3 - c4 - c5 #拆解后再生产的总利润
					
                    #拆解循环的概率
					f_c = f_uq if d else 0
					
                    #最终的总利润
					final_profit = p1 + (p2 * f_c) / (1 - f_c) if f_c else 0
					
                    #选取最优的决策组合
					if (final_profit > max_profit):
						max_profit = final_profit
						best_decision = {
							"零配件1是否检测":"是" if t1 else "否",
							"零配件2是否检测":"是" if t2 else "否",
							"成品是否检测":"是" if t else "否",
							"不合格成品是否拆解":"是" if d else "否"
						}
	return best_decision

#对应所给的6种不同的情况
scenario = [
	[0.1 , 4 , 2 , 0.1 , 18 , 3 , 0.1 , 6 , 3 , 56 , 6 , 5],
	[0.2 , 4 , 2 , 0.2 , 18 , 3 , 0.2 , 6 , 3 , 56 , 6 , 5],
	[0.1 , 4 , 2 , 0.1 , 18 , 3 , 0.1 , 6 , 3 , 56 , 30 , 5],
	[0.2 , 4 , 1 , 0.2 , 18 , 1 , 0.2 , 6 , 2 , 56 , 30 , 5],
	[0.1 , 4 , 8 , 0.2 , 18 , 1 , 0.1 , 6 , 2 , 56 , 10 , 5],
	[0.05 , 4 , 1 , 0.05 , 18 , 3 , 0.05 , 6 , 3 , 56 , 10 , 40]
]

cnt = 1
for [p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d] in scenario:
    print(f"=====第{cnt}种情况=====")
    print("最优决策组合为：")
    best_decision = cal_profits(p1_d,c1_b,c1_t,p2_d,c2_b,c2_t,final_d,c_a,c_t,p,l_change,c_d)
```

# 问题三

根据题目，需要编写函数，分别处理以下四个问题：
1. 零配件是否需要检测
2. 半成品是否需要检测
3. 成品是否需要检测
4. 不合格成品是否需要拆解

```python
import itertools
import pandas as pd
import matplotlib.pyplot as plt
import os
from tabulate import tabulate
import warnings
warnings.filterwarnings('ignore')

# ===================== macOS专属配置：解决中文+图片显示 =====================

# 零配件参数：[次品率, 购买单价, 检测成本]，索引0-7对应配件1-8
parts_params = [
    [0.1, 2, 1],   # 配件1
    [0.1, 8, 1],   # 配件2
    [0.1, 12, 2],  # 配件3
    [0.1, 2, 1],   # 配件4
    [0.1, 8, 1],   # 配件5

    [0.1, 12, 2]   # 配件8
]

# 半成品参数：[标称次品率, 检测成本, 装配成本, 拆解费用]，索引0-2对应半成品1-3
semi_params = [
    [0.1, 4, 8, 6],  # 半成品1（配件1-4）
    [0.1, 4, 8, 6],  # 半成品2（配件5-6）
    [0.1, 4, 8, 6]   # 半成品3（配件7-8）
]

# 成品参数：[标称次品率, 检测成本, 装配成本, 拆解费用, 市场售价, 调换损失]
fin_params = [0.1, 6, 8, 10, 200, 40]

# 半成品对应零配件索引映射
semi2parts = [[0,1,2,3], [4,5], [6,7]]

# ===================== 2. 定义核心计算函数（保留原逻辑） =====================
def calc_part_layer(part_decision):
    """计算配件层：正品率+采购/检测总成本"""
    part_q = []
    part_cost = 0
    for i in range(8):
        p_def, p_buy, p_test = parts_params[i]
        is_test = part_decision[i]
        q = 1.0 if is_test else (1 - p_def)  # 检测则正品率100%
        part_q.append(q)
        part_cost += p_buy + (p_test if is_test else 0)
    return part_q, part_cost

def calc_semi_layer(part_q, semi_decision):
    """计算半成品层：正品率+装配/检测/拆解总成本"""
    semi_q = []
    semi_cost = 0
    for i in range(3):
        p_def, p_test, p_assem, p_disas = semi_params[i]
        is_test, is_disas = semi_decision[2*i], semi_decision[2*i+1]
        # 计算半成品原料正品率（所有对应配件合格）
        part_q_sub = [part_q[j] for j in semi2parts[i]]
        raw_q = 1.0
        for q in part_q_sub:
            raw_q *= q
        # 半成品整体正品率
        semi_raw_q = raw_q * (1 - p_def)
        q = 1.0 if is_test else semi_raw_q
        semi_q.append(q)
        # 计算成本（含拆解期望成本）
        disas_cost = p_disas * (1 - semi_raw_q) if (is_test and is_disas) else 0
        semi_cost += p_assem + (p_test if is_test else 0) + disas_cost
    return semi_q, semi_cost

def calc_fin_layer(semi_q, fin_decision, part_cost, semi_cost):
    """计算成品层：市场合格率+单位期望利润"""
    p_def, p_test, p_assem, p_disas, s_price, s_loss = fin_params
    is_test, is_disas = fin_decision[0], fin_decision[1]
    # 成品原料正品率（3个半成品均合格）
    raw_q = 1.0
    for q in semi_q:
        raw_q *= q
    fin_raw_q = raw_q * (1 - p_def)
    fin_q = 1.0 if is_test else fin_raw_q  # 成品检测则正品率100%
    fin_qualify = fin_q  # 成品市场合格率

    # 成品层成本（装配+检测+拆解+调换损失）
    disas_cost = p_disas * (1 - fin_raw_q) if (is_test and is_disas) else 0
    loss_cost = s_loss * (1 - fin_raw_q) if not is_test else 0  # 不检测才会有调换损失
    fin_cost = p_assem + (p_test if is_test else 0) + disas_cost + loss_cost

    # 总期望成本+利润（利润为负则置为-∞，无生产意义）
    total_cost = part_cost + semi_cost + fin_cost
    profit = max(s_price - total_cost, -float('inf'))
    return fin_qualify, profit

# ===================== 3. 定义帕累托最优筛选函数 =====================
def pareto_optimal(df):
    """筛选帕累托最优解：无其他解的合格率+利润均更优"""
    df = df.sort_values(by=['合格率', '利润'], ascending=[False, False]).reset_index(drop=True)
    pareto_idx = []
    max_profit = -float('inf')
    for idx, row in df.iterrows():
        if row['利润'] > max_profit:
            pareto_idx.append(idx)
            max_profit = row['利润']
    df_pareto = df.loc[pareto_idx].reset_index(drop=True)
    return df_pareto

# ===================== 4. 定义决策编码解读函数（macOS终端友好） =====================
def decode_decision(decision):
    """将16位二进制决策编码解析为中文说明，避免终端乱码"""
    decode = {}
    # 配件层（前8位）
    for i in range(8):
        decode[f'配件{i+1}'] = '检测' if decision[i] == 1 else '不检测'
    # 半成品层（8-13位）
    decode['半成品1检测'] = '是' if decision[8] == 1 else '否'
    decode['半成品1拆解'] = '是' if decision[9] == 1 else '否'
    decode['半成品2检测'] = '是' if decision[10] == 1 else '否'
    decode['半成品2拆解'] = '是' if decision[11] == 1 else '否'
    decode['半成品3检测'] = '是' if decision[12] == 1 else '否'
    decode['半成品3拆解'] = '是' if decision[13] == 1 else '否'
    # 成品层（14-15位）
    decode['成品检测'] = '是' if decision[14] == 1 else '否'
    decode['成品拆解'] = '是' if decision[15] == 1 else '否'
    # 转化为表格格式（tabulate友好）
    decode_list = [[k, v] for k, v in decode.items()]
    return decode_list
```
