# 拉格朗日插值法
## 1. 定义

假设我们有一组已知的数据点：$x_{0},x_{1}​,…,x_{n}$​。

对于其中的第 $i$ 个点 $x_{i}$​，它对应的拉格朗日基函数 $l_{i}​(x)$ 定义为：
$$l_{i}(x)=\prod_{j \neq 1}^ {0\leq j\leq n} \frac{x-x_{j}}{x_{i} - x_{j}}$$
## 2. 这个函数的性质

这个公式看起来复杂，但它的目的非常简单：

**构造一个多项式，使得它在 $x_{i}$ 这一点等于 1，而在其他所有已知点 $x_{j} (j\neq i)$ 都等于 0。**

即：
$$l_{i}(x_{k})=\begin{cases}1~~~if ~k=i \\0~~~if ~k\neq i\end{cases}$$

## 3. 如何用它来插值？

有了这些基函数，我们想要的最终插值多项式 $L(x)$ 就是把它们 “组装” 起来：
$$L(x)=\sum_{i=0}^n y_{i}l_{i}(x)=y_{0}l_{0}(x) + y_{1}l_{1}(x) + \dots+y_{n}l_{n}(x)$$
## Code:
```python
#利用二次多项式进行插值
df_imputed=df_merged.interpolate(method = 'polynomial',order = 2).bfiil().ffill()
```
### 1. 核心原理解释

多项式插值的思路是：**用一个 n 阶多项式函数去拟合已知的数据点，使得该多项式曲线精确地穿过这些已知点，然后利用这个多项式函数来计算缺失位置的值。**

- **与线性插值的区别**：
    
    - 线性插值（`method='linear'`）是一阶多项式，本质是用**直线**连接相邻点。
    - 多项式插值（二阶及以上）是用**曲线**拟合数据。
    
### 2. 关键参数：`order`

你必须指定 `order` 来决定用几次曲线去拟合。

- `order=1`：等同于线性插值（直线）。
- `order=2`：二次多项式（抛物线），可以拟合有一个弯曲的数据。
- `order=3`：三次多项式（立方曲线），可以拟合有两个弯曲的数据。

**对于插值算法无能为力的头部和尾部，使用简单粗暴但有效的 bfill 和 ffill 来兜底.**
#### 补齐开头：`bfill()` (Backward Fill)

- **动作**：**后向填充**。从后往前看，用**后面最近的一个非 NaN 值**来填充前面的 NaN。
- **目的**：专门用来解决**数据开头**的残留 NaN（这是第一步多项式插值搞不定的地方）。
#### 补齐结尾：`ffill()` (Forward Fill)

- **动作**：**前向填充**。从前往后看，用**前面最近的一个非 NaN 值**来填充后面的 NaN。
- **目的**：专门用来解决**数据结尾**的残留 NaN（这也是第一步插值搞不定的地方）。

# K-S正态性检验

1. 独立同分布的样本观测值为 $X_{1},X_{2}​,…,X_{n}$​，样本量为 $n$。
2. 将样本从小到大排序为：$X_{1}​≤X_{2}​\leq \dots\leq X_{n}$​（顺序统计量）。
3. **原假设 H0​**：样本来自某个指定的连续型理论分布，其累积分布函数为 $F_{0}​(x)$（例如标准正态分布 $\phi(x)$）。
4. **备择假设 H1​**：样本不来自该理论分布。

---
## 基础概念：经验分布函数（EDF）

**经验分布函数（Empirical Distribution Function, EDF）** 是对样本真实分布的非参数估计，记为 $F_{n}​(x)$。
### 数学定义
$$F_{n}(x)=\frac{1}{n}\sum _{i=1}^nI\left \{ X_{i}\leq x\right\}$$
其中 $I\{⋅\}$ 是**指示函数**：
$$I\{X_{i}\leq  x\}=\begin{cases}1~~~如果X_{i}\leq x \\0~~~如果X_{i} > x \end{cases}$$
### 直观含义

$F_{n}(x)$ 表示**样本中小于等于 x 的观测值的比例**。它是一个阶梯函数，在每个排序后的样本点 $X_{i}$​ 处跳跃 $\frac{1}{n}$ ​。

---
## 核心：K-S 检验统计量 $D_{n}$ 的构造

K-S 检验统计量 $D_{n}$​ 定义为 **经验分布函数 $F_{n}​(x)$ 与理论分布函数 $F_{0}​(x)$ 在所有 x 上的最大垂直距离**。

### 数学公式
$$D_{n}=sup_{x}|F_{n}(x)-F_{0}(x)|$$

其中 sup 表示**上确界**（可理解为 “最大值”）。
### 等价简化计算（基于顺序统计量）

由于 $F_{n}(x)$ 是阶梯函数，最大距离必然出现在某个样本点 $X_{i}$​ 处。因此，$D_{n}$ 可通过以下更高效的方式计算：
$$D_{n}=max_{1\leq i\leq n}(D_{n}^+ - D_{n}^-)$$

其中：
- 上偏差：$D_{n}^+=max_{1\leq i\leq n}( \frac{i}{n}-F_{0}(X_{i}) )$
- 下偏差：$D_{n}^-=max_{1\leq i\leq n}(F_{0}\left( X_{i})- \frac{i-1}{n} \right)$
### Dn​ 的性质
- 取值范围：$0\leq D_{n}\leq1$
- 含义：$D_{n}$​ 越接近 0，说明样本经验分布与理论分布贴合得越好；Dn​ 越大，偏离越显著。
---
## 原假设下的统计性质（分布自由性）

K-S 检验的核心优势在于其 **分布自由（Distribution-Free）** 特性：

> **关键结论**：在原假设 H0​ 成立（即样本确实来自 $F_{0}​(x)$）且 $F_{0}​(x)$ 是连续分布的前提下，$D_{n}$​ 的抽样分布**不依赖于 $F_{0}(x)$ 的具体形式**。

这意味着无论你检验的是正态分布、均匀分布还是指数分布，只要是连续分布，$D_{n}$ 的临界值表是通用的。

---
## 大样本近似：Kolmogorov 极限定理

当样本量 $n\rightarrow \infty$ 时，$\sqrt{n}D_{n}$​ 依分布收敛于 **Kolmogorov 分布**。
### 极限分布的 CDF

设 K 为服从 Kolmogorov 分布的随机变量，则其累积分布函数为：
$$P\{k\leq x\}=1-2\sum_{k=1}^ \infty (-1)^{k-1}e^{-2k^2 x^2}$$
### 实际应用

在 `scipy.stats.kstest` 中：

- 当 n 较小时，算法会通过精确分布计算 p 值。
- 当 n 较大时，会利用上述 Kolmogorov 极限定理进行近似计算。

**Code:**
```python
ks_stat, p_val = stats.kstest(df_scaled[col], 'norm')
```
---
## 假设检验决策：p 值与临界值

### p 值的定义

p 值是在原假设 $H_{0}$​ 成立的情况下，观察到的 $D_{n}$​ 大于或等于当前计算值 $d_{n}$​ 的概率：
$$p_{value}=P(D_{n}\geq d_{n}|H_{0})$$
### 决策规则（常用显著性水平 α=0.05）

1. **若 $p_{value} >\alpha$**：不拒绝原假设，认为样本与理论分布无显著差异。
2. **若 $p_{value}\leq \alpha$**：拒绝原假设，认为样本不来自该理论分布。
