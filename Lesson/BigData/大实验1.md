### （1）实验目的

- 掌握大规模数据采集方法（网络爬虫 / 公开数据集）。
- 学习数据清洗与预处理技术。
- 熟悉 Hadoop 分布式文件系统（HDFS）的基本操作。
### （2）方法原理

- 使用 Python 的 Requests 库或 Scrapy 框架爬取网页数据，或从 Kaggle、政府开放平台获取结构化 / 半结构化数据。
- 通过 Pandas 处理缺失值、重复值、异常值，并转换为标准格式（如 CSV、JSON）。
- 利用 hadoop fs 命令或 Python 的 hdfs3 库将清洗后的数据上传至 HDFS。
### （3）主要仪器及材料

- 硬件：本地计算机或云服务器（至少 4 核 CPU，8GB 内存）。
- 软件：Hadoop 集群环境、Python 3.x、Jupyter Notebook、库（Pandas、Requests、BeautifulSoup）。
- 数据集：电商用户行为日志、社交媒体文本数据（示例数据集可选用 Amazon Reviews）。
### （4）掌握要点

- 网络爬虫的合法性（Robots 协议）与反爬策略。
- 数据清洗的常用方法（正则表达式、分箱、插值）。
- HDFS 的分布式存储机制与命令行操作。
### （5）实验内容

1. 数据采集：
	-  目标：从 `books.toscrape.com` 抓取书名、价格、评分、是否有库存等，生成 `raw_data.csv`，用 Pandas 清洗生成 `clean_data.csv`，并上传到 HDFS。
	- 真实站点（练习专用）
	- Books to Scrape（练习站）：http://books.toscrape.com/ （**可抓取**，专为爬虫练习设计）
	- 环境与依赖
		- Python 3.8+
		- 建议虚拟环境
2. 数据清洗：
    - 使用 Pandas 去除重复数据、填充缺失值（如用均值或众数）。
    - 提取关键字段（如用户 ID、行为类型、时间戳）。
    - 输出清洗后的 clean_data.csv。
