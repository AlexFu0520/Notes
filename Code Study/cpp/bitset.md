`bitset` 是 C++ 标准库（`<bitset>` 头文件）提供的**固定大小的位序列容器**，专门用于高效处理二进制位操作，相比手动用整数 / 数组模拟位操作，它更简洁、类型安全且提供了丰富的位操作接口。
## 一、核心特性

1. **固定大小**：大小在编译期确定（必须是常量表达式），无法动态调整；
2. **位级存储**：每个元素对应 1 个二进制位（bit），内存利用率极高；
3. **下标访问**：`bitset[n]` 访问第 `n` 位（**0 对应最低位**）；
4. **原生位运算**：支持 `&`、`|`、`^`、`~`、`<<`、`>>` 等位运算，且运算对象需同大小。

## 二、头文件与初始化

### 1. 头文件

```cpp
#include <bitset>  // 必须包含
#include <iostream>
using namespace std;
```

### 2. 常见初始化方式

|初始化方式|说明|示例|
|---|---|---|
|默认初始化|所有位设为 0|`bitset<8> b1;`（00000000）|
|整数 / 无符号数初始化|按二进制填充（高位截断）|`bitset<8> b2(10);`（00001010）|
|字符串初始化|按字符 '0'/'1' 填充（高位在前）|`bitset<8> b3("1010");`（00001010）|
|字符数组初始化|同字符串，需以 '\0' 结尾|`bitset<8> b4("11001010");`|

> 注意：字符串长度小于 bitset 大小时，高位补 0；大于时截断高位。

## 三、常用操作

### 1. 位访问与状态检查

|方法|功能|示例|
|---|---|---|
|`b[pos]`|访问第 pos 位（不检查越界）|`b[0] = 1;`（设最低位为 1）|
|`b.test(pos)`|检查第 pos 位是否为 1（越界抛异常）|`if (b.test(3)) { ... }`|
|`b.any()`|是否至少有 1 位为 1|`bool has1 = b.any();`|
|`b.none()`|是否所有位为 0|`bool all0 = b.none();`|
|`b.all()`|C++11 起，是否所有位为 1|`bool all1 = b.all();`|

### 2. 位修改

| 方法                | 功能                          | 示例                       |
| ----------------- | --------------------------- | ------------------------ |
| `b.set()`         | 所有位设为 1                     | `b.set();`               |
| `b.set(pos, val)` | 设第 pos 位为 val（val 默认为 true） | `b.set(2, 0);`（第 2 位设 0） |
| `b.reset()`       | 所有位设为 0                     | `b.reset();`             |
| `b.reset(pos)`    | 设第 pos 位为 0                 | `b.reset(1);`            |
| `b.flip()`        | 所有位取反                       | `b.flip();`              |
| `b.flip(pos)`     | 第 pos 位取反                   | `b.flip(3);`             |

### 3. 长度与计数

|方法|功能|示例|
|---|---|---|
|`b.size()`|返回总位数（编译期固定）|`int len = b.size();`|
|`b.count()`|返回 1 的位数（汉明重量）|`int cnt1 = b.count();`|

### 4. 类型转换

|方法|功能|注意事项|
|---|---|---|
|`b.to_ulong()`|转换为 unsigned long|超出范围抛 overflow_error|
|`b.to_ullong()`|C++11 起，转换为 unsigned long long|同上|
|`b.to_string()`|转换为 string（高位在前）|如 `b(10)` → "00001010"|

### 5. 位运算

支持 `&`、`|`、`^`、`~`、`<<`、`>>` 及赋值版（`&=`、`|=`、`^=`、`<<=`、`>>=`）：


```cpp
bitset<4> b1(0b1010), b2(0b0110);
bitset<4> b3 = b1 & b2;  // 0010
bitset<4> b4 = b1 | b2;  // 1110
bitset<4> b5 = ~b1;      // 0101（按位取反）
b1 <<= 1;                // b1 变为 0100（左移1位，低位补0）
```

## 四、完整示例

```cpp
#include <bitset>
#include <iostream>
using namespace std;

int main() {
    // 1. 初始化
    bitset<8> b("1010");  // 00001010
    cout << "初始值：" << b << endl;  // 输出 00001010

    // 2. 位修改
    b.set(4);             // 设第4位为1 → 00011010
    b.flip(1);            // 第1位取反 → 00011000
    cout << "修改后：" << b << endl;  // 输出 00011000

    // 3. 状态检查
    cout << "1的位数：" << b.count() << endl;  // 输出 2
    cout << "第3位是否为1：" << boolalpha << b.test(3) << endl;  // true

    // 4. 位运算
    bitset<8> b2(0b00100100);
    bitset<8> b3 = b | b2;
    cout << "b | b2 = " << b3 << endl;  // 00111100

    // 5. 类型转换
    cout << "转字符串：" << b.to_string() << endl;  // 00011000
    cout << "转无符号长整数：" << b.to_ulong() << endl;  // 24

    return 0;
}
```

输出：

```plaintext
初始值：00001010
修改后：00011000
1的位数：2
第3位是否为1：true
b | b2 = 00111100
转字符串：00011000
转无符号长整数：24
```

## 五、典型使用场景

1. **位掩码**：权限控制（如读 / 写 / 执行权限用 3 位表示）、状态标志（如设备在线 / 离线）；
2. **二进制数据处理**：网络协议解析、文件格式（如位图、压缩算法）的位操作；
3. **算法优化**：埃氏筛法（用 bitset 存储素数状态，内存占用仅为数组的 1/8）；
4. **哈希表 / 布隆过滤器**：用位存储元素是否存在，大幅节省内存。
## 六、注意事项

1. `bitset` 的大小必须是**编译期常量**（如 `bitset<8>` 合法，`int n=8; bitset<n>` 非法）；
2. `operator[]` 不做越界检查，直接访问内存，需自行保证 pos 合法；
3. 转换为整数时（`to_ulong()`/`to_ullong()`），若 bitset 数值超出目标类型范围，会抛 `std::overflow_error`。
