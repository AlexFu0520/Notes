---

本仓库是个人 Obsidian 知识库的全量备份，沉淀每日任务记录、算法学习、代码研究、课程笔记与刷题复盘等核心内容，以日为单位持续迭代，完整记录个人学习成长的全轨迹。

---
## 仓库概览

- 仓库类型：Obsidian Vault 全量备份
- 核心内容：每日任务 / 日记、算法模板、代码学习、课程笔记、刷题记录
- 更新频率：日常高频更新，核心内容按日同步

---

## 目录结构说明


| 文件夹                                         | 核心用途                                                |
| ------------------------------------------- | --------------------------------------------------- |
| `.obsidian`                                 | Obsidian 软件核心配置目录，包含插件、主题、快捷键、工作区等 Vault 环境全量配置     |
| `2025 Daily Task`                           | 2025 年度每日任务、日记、日常复盘与成长记录全归档                         |
| `2026 Daily Task`                           | 2026 年度每日任务、日记、当日复盘、计划安排核心目录，按日更新                   |
| `Algorithm Template` / `algorithm template` | 算法竞赛 / 刷题常用模板代码、知识点总结、速查手册，按算法专题分类归档                |
| `Code Study`                                | 代码学习、源码解读、项目实践、技术研究笔记，配套图片资源存放于 `code study/images` |
| `Excalidraw`                                | 基于 Excalidraw 插件绘制的技术原理图、知识图谱、流程图、手绘笔记等             |
| `Lesson`                                    | 专业课程、网课、培训学习的随堂笔记、知识点梳理、重点难点复盘                      |
| `题目`                                        | 算法刷题、课后习题、真题训练的题目记录、解题思路、代码实现、错题深度复盘                |
